Свойства CSS (Cascading Style Sheets) — это набор инструкций, определяющих внешний вид элементов HTML. Каждое свойство отвечает за конкретный аспект стилизации. Вот основные категории и примеры свойств CSS: Свойства для работы с текстом: color — цвет текста, font-size — размер шрифта, font-family — шрифт текста, text-align — выравнивание текста, line-height — высота строки. Свойства для работы с блоками и размерами: width — ширина элемента, height — высота элемента, margin — внешний отступ, padding — внутренний отступ, border — граница элемента.
